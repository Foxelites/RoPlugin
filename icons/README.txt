ICON INSTRUCTIES VOOR ROSTREAMER
=====================================

Plaats hier je plugin icoontjes met de volgende namen en groottes:

1. icon16.png  - 16x16 pixels  (gebruikt in de extensiebalk)
2. icon48.png  - 48x48 pixels  (gebruikt in de extensie manager)
3. icon128.png - 128x128 pixels (gebruikt in de Chrome Web Store)

TIPS:
- Gebruik PNG formaat met transparante achtergrond
- Zorg dat het design duidelijk zichtbaar is op kleine formaten
- Gebruik dezelfde kleuren als je plugin (#00b06f groen past goed!)
- Maak een simpel, herkenbaar design (bijv. een Robux symbool met een oog of blur effect)

ONLINE TOOLS OM ICONS TE MAKEN:
- Canva.com (gratis, makkelijk)
- Photopea.com (gratis Photoshop alternatief)
- Figma.com (gratis voor basics)

Je kunt ook online "icon resizer" tools gebruiken om één groot icoon te verkleinen naar de 3 groottes.
